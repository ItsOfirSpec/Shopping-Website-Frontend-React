# Project Readme

קובץ זה נועד להסביר כיצד להפעיל, לערוך ולהשתמש בפרויקט.

---

## דרישות מערכת

לפני ההפעלה, ודא שהמערכת שלך עומדת בדרישות הבאות:

- Node.js ≥ 18
- npm ≥ 9

---

## הפעלה

כדי להפעיל את הפרויקט:

1. פתח את תיקיית הפרויקט.
2. הרץ את הקובץ `StartNpm.bat`.

הקובץ `StartNpm.bat` יוודא אם כל התלויות מותקנות (`node_modules`).  
אם חסר משהו — הוא יבצע התקנה אוטומטית.  
אם הכול תקין — הוא יפעיל את הפרויקט מיד.

---

## עריכת הקוד

במידה ואתה משתמש ב־Visual Studio Code, ניתן לפתוח את הפרויקט בקלות באמצעות הקובץ הבא:

- `StartVisualCode.bat` - מפעיל את Visual Studio Code מתוך תיקיית הפרויקט.

---

## הערות נוספות

אם משהו לא עובד, בדוק:

- שהגרסאות של Node ו־npm תואמות לדרישות.

בהצלחה.
